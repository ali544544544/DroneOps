# 🚁 DRONE OPS

DRONE OPS ist ein leichtgewichtiges GitHub-Pages-Dashboard für Drohnenpiloten.

## Features

- Dashboard mit aktuellem Standort oder gespeicherten Locations
- Golden Hour heute mit SVG-Sonnenbahn
- Flugscore abhängig vom aktiven Drohnenprofil
- Locations-Verwaltung mit Suche, GPS, Karte, Route, Notizen und Logbuch
- Drohnen-Seite mit 7 festen Profilen
- Packliste mit Kategorien, Fortschritt, Bearbeiten und Reset
- DE/EN Umschaltung
- localStorage für Daten, Cache und Einstellungen
